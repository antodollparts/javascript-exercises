# JavaScript Exercises

Repositorio de práctica de JavaScript, creado como parte de mi aprendizaje en el módulo JavaScript desde Cero. Aquí encontrarás ejercicios y experimentos con los conceptos básicos de JS.

---

## 🎯 Objetivo

Practicar los conceptos fundamentales de JavaScript vistos en clase, incluyendo tipos de datos, variables, operadores, estructuras de control y más. Cada actividad se documenta en archivos `.js` independientes.

---

## 📁 Estructura del repositorio

```
javascript-exercises/
├── README.md
├── tipos-de-dato.js       # Tipos de datos en JavaScript (typeof)
└── comparacion-notas.js   # Comparación y decisiones en JavaScript
```


---

## 🛠️ ¿Cómo ejecutar los archivos?

### Opción 1 — Node.js (terminal)
```bash
node nombre-archivo.js
```

### Opción 2 — Navegador (consola)
1. Abre el navegador y presiona `F12`
2. Ve a la pestaña **Consola**
3. Copia y pega el contenido del archivo
